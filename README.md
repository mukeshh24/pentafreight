# 🚀 Enhancements & Improvements

While recreating the provided design, I identified a few opportunities to improve the overall user experience. The implementation remains faithful to the original design while introducing small frontend enhancements that improve usability and performance.

### Improvements Implemented

- Improved image loading experience in areas where images on the reference website occasionally load with noticeable delay or fail to appear immediately.
- Enhanced responsive behavior for the **Services** section.
  - On screens **768px and above**, the original stacked GSAP animation has been recreated.
  - On screens **below 768px**, the animation is replaced with a cleaner static background layout to ensure better readability, usability, and overall mobile performance.
- Added **draggable interaction** to the Industry cards, allowing users to freely drag and explore the cards for a more engaging experience.
- Enhanced the **Awards** section by implementing a hover interaction where the active award remains highlighted while the remaining awards subtly blur, creating a stronger visual focus.
- Built reusable, JSON-driven components for better scalability and easier maintenance.
- Optimized animations and component structure to improve overall frontend performance and maintainability.

### Notes

The objective of this assignment was to recreate the provided pages as accurately as possible while making small frontend improvements that enhance usability without changing the overall design language.
